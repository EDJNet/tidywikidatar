# A zero-rows tibble used internally when `tw_get_qualifiers()` would not return any value.

A zero-rows tibble used internally when
[`tw_get_qualifiers()`](https://edjnet.github.io/tidywikidatar/reference/tw_get_qualifiers.md)
would not return any value.

## Usage

``` r
tw_empty_qualifiers
```

## Format

A data frame with 0 rows and 8 columns
