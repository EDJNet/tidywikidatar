# The Wikidata Q identifier of all members of the European Parliament since its establishment

A dataset with all the Wikidata items that have "Q27169" (member of the
European Parliament) for the property "P39" (position held).

## Usage

``` r
tw_qid_meps
```

## Format

A data frame with 4581 rows and 1 column:

- id:

  Q identifiers
