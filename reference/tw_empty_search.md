# A zero-rows tibble used internally when `tw_search()` would not return any value.

A zero-rows tibble used internally when
[`tw_search()`](https://edjnet.github.io/tidywikidatar/reference/tw_search.md)
would not return any value.

## Usage

``` r
tw_empty_search
```

## Format

A data frame with 0 rows and 4 columns
