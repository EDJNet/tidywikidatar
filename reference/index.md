# Package index

## Core functions used to get data from Wikidata as data frames

- [`tw_get()`](https://edjnet.github.io/tidywikidatar/reference/tw_get.md)
  : Return (most) information from a Wikidata item in a tidy format
- [`tw_get_property()`](https://edjnet.github.io/tidywikidatar/reference/tw_get_property.md)
  : Get Wikidata property of one or more items as a tidy data frame
- [`tw_get_qualifiers()`](https://edjnet.github.io/tidywikidatar/reference/tw_get_qualifiers.md)
  : Get Wikidata qualifiers for a given property of a given item
- [`tw_get_p_wide()`](https://edjnet.github.io/tidywikidatar/reference/tw_get_p_wide.md)
  : Efficiently get a wide table with various properties of a given set
  of Wikidata identifiers

## Core functions that return character vectors of the same length as input

These are commonly used with piped operations.

- [`tw_get_label()`](https://edjnet.github.io/tidywikidatar/reference/tw_get_label.md)
  : Get Wikidata label in given language
- [`tw_get_description()`](https://edjnet.github.io/tidywikidatar/reference/tw_get_description.md)
  : Get Wikidata description in given language
- [`tw_get_property_same_length()`](https://edjnet.github.io/tidywikidatar/reference/tw_get_property_same_length.md)
  [`tw_get_p()`](https://edjnet.github.io/tidywikidatar/reference/tw_get_property_same_length.md)
  : Get Wikidata property of an item as a vector or list of the same
  length as input
- [`tw_get_p1()`](https://edjnet.github.io/tidywikidatar/reference/tw_get_p1.md)
  : Get Wikidata property of an item as a character vector of the same
  length as input
- [`tw_get_property_label()`](https://edjnet.github.io/tidywikidatar/reference/tw_get_property_label.md)
  : Get label of a Wikidata property in a given language
- [`tw_get_property_description()`](https://edjnet.github.io/tidywikidatar/reference/tw_get_property_description.md)
  : Get description of a Wikidata property in a given language

## Labelling

Besides the dedicated functions for labelling,
[`tw_label()`](https://edjnet.github.io/tidywikidatar/reference/tw_label.md)
attempts to turn Q identifiers to labels in data frames such as the ones
generated by the core functions listed above.

- [`tw_label()`](https://edjnet.github.io/tidywikidatar/reference/tw_label.md)
  : Gets labels for all columns with names such as "id" and "property".

## Filters

These are convenience functions; the result is similar to using
[`tw_get()`](https://edjnet.github.io/tidywikidatar/reference/tw_get.md)
and
[`dplyr::filter()`](https://dplyr.tidyverse.org/reference/filter.html),
but with less typing.

- [`tw_filter()`](https://edjnet.github.io/tidywikidatar/reference/tw_filter.md)
  : Filter search result and keep only items with matching property and
  Q identifier
- [`tw_filter_first()`](https://edjnet.github.io/tidywikidatar/reference/tw_filter_first.md)
  : Filter search result and keep only and keep only the first match
- [`tw_filter_people()`](https://edjnet.github.io/tidywikidatar/reference/tw_filter_people.md)
  : Filter search result and keep only people

## Search Wikidata

  label, and description.
- [`tw_search_item()`](https://edjnet.github.io/tidywikidatar/reference/tw_search_item.md)
  : Search for Wikidata properties in Wikidata and return Wikidata id,
  label, and description.
- [`tw_search_property()`](https://edjnet.github.io/tidywikidatar/reference/tw_search_property.md)
  : Search for Wikidata properties in Wikidata and return Wikidata id,
  label, and description.
- [`tw_search_single()`](https://edjnet.github.io/tidywikidatar/reference/tw_search_single.md)
  : Search for Wikidata items or properties and return Wikidata id,
  label, and description.

## Functions used internally to check for validity of data

They are exported for easier access to advanced users, but should be
disregarded by most.

- [`tw_check_qid()`](https://edjnet.github.io/tidywikidatar/reference/tw_check_qid.md)
  : Ensures that input appears to be a valid Wikidata id
- [`tw_check_pid()`](https://edjnet.github.io/tidywikidatar/reference/tw_check_pid.md)
  : Ensures that input appears to be a valid Wikidata property id (i.e.
  it starts with P and is followed only by digits)
- [`tw_check_search()`](https://edjnet.github.io/tidywikidatar/reference/tw_check_search.md)
  : Checks if an input is a search; if not, it tries to return a search
