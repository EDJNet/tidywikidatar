# A zero-rows tibble used internally when `tw_get_wikipedia_page_qid()` would not return any value.

A zero-rows tibble used internally when
[`tw_get_wikipedia_page_qid()`](https://edjnet.github.io/tidywikidatar/reference/tw_get_wikipedia_page_qid.md)
would not return any value.

## Usage

``` r
tw_empty_wikipedia_page
```

## Format

A data frame with 0 rows and 6 columns
