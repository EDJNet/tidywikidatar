# A zero-rows tibble used internally when `tw_get_image_metadata()` would not return any value.

A zero-rows tibble used internally when
[`tw_get_image_metadata()`](https://edjnet.github.io/tidywikidatar/reference/tw_get_image_metadata.md)
would not return any value.

## Usage

``` r
tw_empty_image_metadata
```

## Format

A data frame with 0 rows and 19 columns
