# A zero-rows tibble used internally when `tw_get_wikipedia_page_sections()` would not return any value.

A zero-rows tibble used internally when
[`tw_get_wikipedia_page_sections()`](https://edjnet.github.io/tidywikidatar/reference/tw_get_wikipedia_page_sections.md)
would not return any value.

## Usage

``` r
tw_empty_wikipedia_page_sections
```

## Format

A data frame with 0 rows and 8 columns
