# The Wikidata Q identifier of European airports found in Eurostat's `avia_par_` dataset

The Wikidata Q identifier of European airports found in Eurostat's
`avia_par_` dataset

## Usage

``` r
tw_qid_airports
```

## Format

A data frame with 429 rows and 1 column:

- id:

  Q identifiers
