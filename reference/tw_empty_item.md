# A zero-rows tibble used internally when `tw_get()` would not return any value.

A zero-rows tibble used internally when
[`tw_get()`](https://edjnet.github.io/tidywikidatar/reference/tw_get.md)
would not return any value.

## Usage

``` r
tw_empty_item
```

## Format

A data frame with 0 rows and 3 columns
